## Bugs fixed (real, shipped + verified)

1. **`crossproject.detect_conflicts` missing `return warnings`** (commit 07cd403)
   Function built its result list but fell off the end, so every call
   returned None. Caught while writing tests; no callers yet so it
   never fired in production.
2. **`BaseImageViewer.set_path` AttributeError on real files** (commit 849c787)
   Called `.isNull()` on the tuple `(QPixmap, w, h)` returned by
   `load_pixmap()`. Crashed every successful path load. No callers
   yet (planned PreviewPane migration), caught preemptively.
3. **`oneup.mcp_init_session` / `mcp_tool_call` socket leak** (commit 10e1bf0)
   Both called `urlopen()` and used the response without closing.
   Long sessions of OneUp MCP calls leaked file descriptors. Wrapped
   in `with` blocks.
4. **PIL `Image.open(...).convert(...)` source-file FD leak**
   (commits 7a80bd9, 2d332df, 36871a8)
   Pillow's lazy load held source files open until GC. Wrapped 8
   sites across exporter, composer_right, __main__ CLI, imaging.
   On Windows this also unblocks source-file moves/deletes mid-batch.
5. **`strategy._section_brand_notes` unreachable branch** (commit 2d3e2e4)
   `if len(lines) == 1: lines.append("No identity configured")` was
   dead because the function always pre-fills voice line. Removed.
