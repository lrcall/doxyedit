# Keyboard Shortcuts

---

## File Operations

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New project |
| Ctrl+O | Open project |
| Ctrl+S | Save project |
| Ctrl+Shift+S | Save As |
| Ctrl+E | Export all platforms |
| Ctrl+V | Paste image / file path / folder path / URL |
| F5 | Reload project from disk (picks up external edits) |
| Shift+F5 | Force thumbnail recache |

---

## Selection (Assets Tab)

| Shortcut | Action |
|----------|--------|
| Click | Select one image |
| Ctrl+Click | Toggle multi-select |
| Shift+Click | Range select |
| Alt+Click | Send to Censor tab |
| Ctrl+A | Select all |
| Ctrl+D | Deselect all |
| Escape | Deselect all / clear active tag bar filters |

---

## Navigation (Assets Tab)

| Shortcut | Action |
|----------|--------|
| Up arrow | Previous thumbnail (auto-scrolls into view) |
| Down arrow | Next thumbnail (auto-scrolls into view) |
| Enter | Open preview for selected thumbnail |
| Ctrl+Scroll | Zoom thumbnail grid (80px – 320px) |
| Ctrl+F | Focus search box |

---

## Navigation (Inside Preview Window)

| Shortcut | Action |
|----------|--------|
| Space / Tab / Down | Next image |
| Backspace / Up / Left | Previous image |
| Esc | Close preview |
| Ctrl+0 | Fit image to view |
| N | Add note annotation |
| V | Toggle View Notes |

---

## Tag Shortcuts (Assets Tab)

Default number-key bindings — toggle the tag on all selected images:

| Key | Tag |
|-----|-----|
| 1 | Page / Panel |
| 2 | Character Art |
| 3 | Sketch / WIP |
| 4 | Game Asset |
| 5 | Merch Source |
| 6 | Reference |
| 7 | Final / Approved |
| 8 | Work in Progress |
| 0 | Ignore / Skip |

Custom shortcuts can be assigned to any tag via right-click → **Set Shortcut Key**.

## Asset Actions

| Shortcut | Action |
|----------|--------|
| Delete | Soft-delete (tags as "ignore", hides from grid) |
| Alt+A | Add tag to selected assets |
| Ctrl+H | Temporary hide selected (Ctrl+H with nothing selected restores all) |
| Shift+E | Notes overlay popup (centered on screen) |
| F2 | Rename selected file on disk |
| Ctrl+Shift+C | Copy full file path to clipboard |
